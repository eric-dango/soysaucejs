Carousels

Full list of options

  • peek - shows images on left and right
  • finite - does not repeat
  • autoscroll - timer based forward scrolling
  • noswipe - user cannot swipe
  • 3d - force 3d, off on androids by default
  • zoom - allows the user to zoom in on an item
