What is Soysauce?

Soysauce is a customizable widget library made to ease the life of a developer. If a widget is difficult to implement, Soysauce is doing something wrong! (Psst... just report an issue if you want something changed)

Grab the latest! (v1.1.85)

Soysauce requires jQuery. To install, just add the CSS and a JS file to your site. Soysauce supports both http and https protocols.

"How To" demo - It's easy, just add tags!

In this demo, we will be creating a widget that autofills the city and state (pretend you're working on a form). This is your HTML without tags. Click the button when you're ready to add the required tags.

<div>
  <input type="tel" placeholder="Zip Code">
  <input type="text" placeholder="City">
  <select>
    ...
  </select>
</div>

Boom. That's it! Now, type in a zip code and wait for the city/state to populate.

<div data-ss-widget="autofill-zip">
  <input data-ss-component="zip" type="tel" placeholder="Zip Code">
  <input data-ss-component="city" type="text" placeholder="City">
  <select data-ss-component="state">
    ...
  </select>
</div>
For this widget, the required attributes are the "data-ss-" prefixed attributes. If you'd like to see more demos, please checkout the demos section!
