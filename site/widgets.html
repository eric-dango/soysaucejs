Carousel

Description

Carousels are simply fancy display containers. They usually hold images, but can be used for content (such as this API section).

Required Attributes

  • data-ss-widget="carousel" - Add to the container
  • data-ss-component="item" - Add to each item you want to display

Optional Attributes

  • data-ss-options="option1 option2 option3" - Add to the container
  • data-ss-component="item" - Add to each item you want to display
