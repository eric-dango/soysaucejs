What is Soysauce?

Soysauce is a customizable widget library made to ease the life of a developer. If a widget is difficult to implement, Soysauce is doing something wrong! (Psst... just report an issue if you want something changed)

Made by branders, for branders. No more outside widgets; we create it for ourselves.

No more project-specific fixes/widgets. Re-usable, scalable javascript.

Have an idea for a widget? No one's stopping you. Create it!
